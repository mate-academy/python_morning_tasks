**Безнульове число** це додатне ціле число, яке **не містить** `0` у своєму десятковому представленні.

На вхід дається ціле число `num`. Поверни _список із двох цілих чисел [a, b], де_:
- `a` і `b` є **Безнульовими числами**.
- `a + b = num`
