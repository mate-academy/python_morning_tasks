Given an integer `number`, return any list containing `number` of **unique** integers such that they add up to `0`.

**_Examples_**:
```python
Input: number = 5
Output: [-7, -1, 1, 3, 4]
Explanation: These arrays also are accepted [-5, -1, 1, 2, 3], [-3, -1, 2, -2, 4]

Input: number = 3
Output: [-1, 0, 1]

Input: number = 1
Output: [0]
```
---
**_Constraints_**:
- 1 <= `number` <= 1000
