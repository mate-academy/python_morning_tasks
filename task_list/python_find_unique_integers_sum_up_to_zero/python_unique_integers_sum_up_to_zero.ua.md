На вхід дається ціле число `number`, поверни будь-який список, що містить `number` **унікальних** цілих чисел, сума яких дорівнює `0`.

**_Приклади_**:
```python
Input: number = 5
Output: [-7, -1, 1, 3, 4]
Пояснення: Тако валідними є наступні списки: [-5, -1, 1, 2, 3], [-3, -1, 2, -2, 4]

Input: number = 3
Output: [-1, 0, 1]

Input: number = 1
Output: [0]
```
---
**_Обмеження_**:
- 1 <= `number` <= 1000
